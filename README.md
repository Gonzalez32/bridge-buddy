# Bridge Buddy - Combat Misinformation App

Bridge Buddy is a comprehensive web application designed to combat misinformation and promote critical thinking in the digital age. Our mission is to provide users with reliable tools to verify information and understand the current landscape of misinformation.

## 🎯 Mission

In an era where misinformation spreads faster than truth, Bridge Buddy serves as your reliable companion in navigating the complex landscape of information. We believe that an informed citizenry is the foundation of a healthy democracy, and we're committed to providing the tools and resources needed to combat harmful narratives and promote critical thinking.

## ✨ Features

### 🔍 Fact Checker
- Submit claims and statements for verification
- Get instant fact-checking results with confidence levels
- Access detailed explanations and reliable sources
- Track your fact-checking history

### 📚 Educational Resources
- Comprehensive media literacy guides
- Critical thinking resources
- Links to reliable fact-checking organizations
- Tips for spotting misinformation

### 📊 Advanced Analytics Dashboard
- Real-time insights into misinformation trends
- User behavior analytics
- Trending topics analysis
- Public data export for researchers and journalists

### 🌐 Community Features
- Share reliable information with your network
- Access community-curated resources
- Support fellow truth-seekers

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/bridge-buddy-app.git
cd bridge-buddy-app
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 📊 Analytics System

Bridge Buddy includes a comprehensive analytics system that tracks:

### User Behavior
- Page views and navigation patterns
- Fact-check submissions and results
- Session duration and engagement
- Resource usage

### Misinformation Trends
- Most common false claims
- Trending topics in fact-checks
- Confidence levels in verification results
- Temporal patterns in misinformation

### Public Data Sharing
- Anonymous analytics data available for researchers
- JSON export functionality
- API endpoints for data access
- Insights reports for journalists

### Privacy & Ethics
- All data is anonymized
- No personal information is collected
- Users can opt out of analytics
- Data is used only for improving the service and research

## 🛠️ Technical Stack

- **Frontend**: React 18 with modern hooks
- **Styling**: Bulma CSS framework
- **Routing**: React Router v6
- **Analytics**: Custom analytics service with localStorage
- **State Management**: React hooks and context
- **Build Tool**: Create React App

## 🔧 Configuration

### Environment Variables
```bash
REACT_APP_API_URL=https://api.bridgebuddy.app
REACT_APP_ANALYTICS_ENABLED=true
REACT_APP_FACT_CHECK_API_KEY=your_api_key
```

### Analytics Configuration
The analytics system can be configured in `src/services/analytics.js`:
- Data retention periods
- Event tracking preferences
- Export formats
- Privacy settings
